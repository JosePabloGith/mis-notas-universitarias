# gia de atajos Rapidos para el explorador de archivos Nvim-Tree

Usamos estos atajos cuando tengamos el cursor posicionado sobre el arbol/explorador de archivos, no funcionan en el buffer principal

---

## 1. Crear , Renombrar y Eliminar

| Tecla | Acción | Descripción |
| :---: | :---: | :--- |
| **`a`** | **A**dd (Añadir) | crea un nuevo archivo o carpeta. (Para carpetas, añade una barra al final, como por ejemplo: `nueva_carpeta/`). |
| **`r`** | **R**ename (Renombrar) | renombra el archivo o carpeta seleccionado. |
| **`d`** | **D**elete (Eliminar) | elimina el archivo o carpeta seleccionado. |
| **`y`** | **Y**ank (Copiar) | copia el archivo o carpeta seleccionado. |
| **`x`** | **C**ut (Cortar) | corta el archivo o carpeta seleccionado. |
| **`p`** | **P**aste (Pegar) | pega el archivo o carpeta copiado o cortado. |

--

## 2. Navegación y Visualización

| Tecla | Acción | Descripción |
| :----: | :---: | :--- |
| **`j` / `k`** | **J/K** | mueve el cursor hacia abajo/arriba en el árbol de archivos. |
| **`Enter`** | Abrir / Expandir | abre el archivo seleccionado o expande la carpeta. |
| **`H`** | Ocultos | Alterna la visualización de archivos ocultos. |
| **`R`** | **R**efresh (Actualizar) | actualiza el árbol de archivos para reflejar cambios recientes. |
| **`q`** | **Q**uit (Salir) | cierra el explorador de archivos. |
